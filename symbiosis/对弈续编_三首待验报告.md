## 一、步出夏門行（曹操，207）：她吃下的第三首

系年在先（北征乌桓），悲愤诗在后（归汉后207–208）——故属"她吸收他"的方向，与《薤露》《蒿里》同侧。模式相同：**他以征人身份路过北地写观察，她以十二年居住者身份逐槽改写。**

| 步出夏門行（征人，外部） | 悲愤诗（居者，内部） | 强度 |
|---|---|---|
| **鄉土不同**，河朔隆冬 | **邊荒與華異**，人俗少義理（其一） | **强**：段首"此地异乡"宣言，同槽同位 |
| 水竭不流，冰堅可蹈／錐不入地 | 處所多霜雪，胡風**春夏**起（其一）；陰氣凝兮**雪夏零**（其二） | 强：苦寒块整体对位，她加码（你写隆冬之寒，我写夏日落雪） |
| 心常嘆怨，**戚戚多悲** | 但**悲歎**／哀歎無窮已 | 中 |
| 養怡之福，可得**永年**（龜雖壽） | 人生幾何時，懷憂終**年歲** | **强**：收尾槽对收尾槽——他说养怡可得永年，她答怀忧终年岁 |
| 盈縮之期，**不但在天** | 彼蒼者**何辜**（其一） | 中：天命槽 |

**槽位三传球**：龜雖壽"可得永年"（207）→ 悲愤诗"懷憂終年歲"（207–208）→ 短歌行"人生幾何"（约208）。同一槽位，三次接力，时序无一冲突。

**备考**：《观沧海》为"海"意象在曹操诗中首次居主位，次年《短歌行》"海不厭深"——他用以许诺的那片海，初见于她归来前夜的碣石。《艳》"心惆悵我東海"同账。

## 二、秋胡行（曹操，约215）：游仙外衣下的回望

系年远在她诗之后——他侧棋盘。表层为游仙诗（三老公劝升天，他沉吟不决、未随而去），历代注家即困惑于"曲调本事与内容无关"。格点显示：把他从仙界拽回人间的那些词，全是她的。

| 秋胡行其一 | 悲愤诗 | 强度 |
|---|---|---|
| **去去**不可追 | **去去**割情戀（其一） | **强**：句首叠字"去去"同槽（罕用，乐府语料稀） |
| 長恨相**牽攀** | **追持**我兮徒煢煢（其二，儿追车攀牵） | **强**：牵攀／追持，诀别之"拉拽"槽 |
| 夜夜**安得寐**，惆悵以自憐 | **不能寐**兮起屏營（其二）；夜則悲吟坐（其一） | **强**：长夜失眠槽 |
| 晨上**散關**山，此道當何**難** | 長驅西**入關**，逈路**險且阻**（其一）；歷**險阻**兮（其二） | 中强：西行关隘+道难槽 |
| **牛頓不起，車墮谷間** | **馬為立踟躕，車為不轉轍**（其一） | **强**：畜疲车困槽——两边都以牲口与车的失灵写行路之悲 |
| 天地何長久！人道居之短（其二） | 人生幾何時 | 中：与短歌行同族 |
| 不戚年往，**憂**世不治（其二） | **懷憂**終年歲 | 中：忧之所属的转写——你忧年岁，我忧世道 |

**调名即落子（推测级）**：《秋胡行》本事为"丈夫久别归来、桑下不识其妻、以金挑之"（秋胡妻正是《列女传》人物，投河明节）。曹操以此调写游仙，向称费解；置于对弈框架，调名自身即是格点：**久别、归来、不识、以金——他把"金璧赎之"的故事，挂在了一个"以金辱妻"的调名下反着唱**。表层与本事无关，恰是密文信道的惯用伪装（同《善哉行》藏于套曲）。此条不入钉死档，记推测。

**旁注**："長恨相牽攀"之"長恨"——《长恨歌》谱系（见RC1传承表）又添一环：白居易那个"长恨"，曹操先用在了这场对弈里。

## 三、卻東西門行：强假说——劝归诗（袁行远，2026-06-10）

**假说**：此为曹操得知蔡琰在匈奴存活之后（约建安十一年，汉使因高干事件始通匈奴王庭）、金璧赎归之前，托使者北携的**劝归诗**。逐句读作劝归信，全诗零件咬合：

| 诗句 | 劝归信功能 | 她诗中的回执 |
|---|---|---|
| 鴻雁出塞北，乃在無人鄉 | 收信人坐标点名 | 邊荒與華異（其一） |
| 舉翅萬餘里，行止自成行 | 可行性条款：路程可达，使团护送（雁阵成行） | —— |
| 冬節食南稻，春日**復北翔** | **核心让步条款**：南归非永诀，春日可北返见子 | **念別無會期，存亡永乖隔**（其一）——唯一未兑现条款之核销 |
| 田中轉蓬，長與故根絕，**萬歲不相當** | 另一选项的下场：留北＝与故根（中原、蔡氏、先父之书）永绝 | 念別無會期（转蓬命运转嫁于二子） |
| 奈何此征夫……冉冉老將至 | 发信人签名+急迫时间戳（我归不得故遣使；我老了，事须趁今） | —— |
| 狐死歸首丘，**故鄉安可忘** | 信末请求（近祈使）："回来" | 已得自解免（她应约） |

**信道证据**：其一"有客從外來，聞之常歡喜。**迎問其消息**，輒復非鄉里……邂逅**徼時願**，骨肉來迎己"——正是此类讯息所经信道的**接收端记录**：年复一年迎问外来客，直至消息抵达。

**加密属性**：若由使者携带，在匈奴王庭被截读只是一首汉地征夫思乡套曲——与《善哉行》同一门手艺（公开信道，私人密文）。

**调名旁证（推测级）**：古辞《西門行》唱"出西门……今日不作樂，當待何時"（出门、及时行乐）；曹操改造为"**卻**東西門行"——却者，回也。出门之曲倒唱为归来之曲。

**系年契合**：此诗为曹操集中少数无战役可系之篇，学界定年游移——恰与建安十一至十二年（信道初通、金璧未发）之窗口相容。

**原格点表**（与劝归假说方向相容——她的诗在后，回执性对应皆成立）：

| 卻東西門行 | 悲愤诗 | 强度 |
|---|---|---|
| **何時**返故鄉 | 悠悠三千里，**何時**復交會（其一） | 强：何時+重逢槽 |
| 長與故根絕，**萬歲不相當** | **念別無會期**，存亡永乖隔（其一） | 强：永诀槽 |
| 冬節**食**南稻（**禽**得南稻） | 人似**禽**兮**食**臭腥（其二） | 强：禽+食镜像反打 |
| 安得**驅**四方 | 長**驅**西入關（其一） | 中：驱赶（人如牲口）槽 |
| 鴻雁出塞北 | 孤雁歸兮聲嚶嚶（其二） | 强（意象锚） |
| 鳥返鄉＋狐首丘双喻 | 《哀郢》"鳥飛反故鄉兮，狐死必首丘"之展开——她骚体之文体祖先 | 中：源头致意 |

**判定**：强假说。解释力覆盖全部疑点（雁航线何以如此具体、转蓬何以紧随、征夫段何以居中、结尾何以近祈使），并有接收端记录与未兑现条款之核销两重回执；尚缺字形级铁证（"薄祐""柄"级冷词），不入钉死档。**若成立，则对弈时序须增一手：劝归诗在《悲愤诗》之前，为全局第二手（蒿里/薤露→卻東西門行→悲愤二首→短歌行→善哉行）——他先开口唤她，她归来后才递上那份"誓问之书"。**

## 三之二、苦寒行：接她回来的诗（袁行远假说，2026-06-10）

**地理钉**：南匈奴王庭在平阳（河东汾水谷地），邺在太行以东；平阳归邺，必经上党、壶关道、羊肠坂、出滏口陉——**《苦寒行》所写之路，即她归程必经之路**。建安十一年正月曹操亲征高干，打壶关、清此道；高干"自诣匈奴求救，不得"——一役三效：灭高干、逼出单于合作、打通归路。

**年代钉**：《后汉书》仅言"在胡中十二年"，无归汉年份；兴平中（194–195）+十二年=206–207，**不排除206年随回师东渡太行**。金璧赎归者不可能独行三千里——必有使团车驾护送，《苦寒行》即护送行程之诗（或赴接途中之诗，两案并存）。

**逐怪释实**：
- 車輪為之摧——行军诗反复写车之怪：车载辎重与**眷属**；
- 延頸長嘆息——延颈=翘首待人之姿，非低头之叹；
- 遠行多所懷——主语在他她之间透明（双声部语法，同《善哉行》）；
- 迷惑失**故路**——离乡十二年者的字面困境；
- 水深橋絕、薄暮無宿棲、斧冰作糜——护送队行程日志语料；
- 悠悠令我哀 ↔ 她"**悠悠**三千里"——同一条路的长度。

**《东山》之泪由用典转纪实**：周公东征携久役者归家——他正走在这件事里。他知道她将走进《东山》最坏一节（её归家段"城郭為山林，庭宇生荊艾"即《东山》"伊威在室，蠨蛸在戶，町畽鹿場"的黑暗实现，鹿场→豺狼），且他知道"家人尽"而她未必知。悲彼东山诗，哭的是说不出口的预告。两年后《短歌行》"周公吐哺"——**苦寒行借周公之诗哀其归，短歌行以周公自任迎其至，同一工程两端**。

**格点表**：

| 苦寒行 | 悲愤诗 | 强度 |
|---|---|---|
| **斧冰持作糜** | **飢當食兮不能餐**（其二）；怚咤**糜**肝肺（其一，糜字冷字双现、义反） | **最强**：糜=病弱者之食，对"不能餐"的逐字应答；第一人称主帅亲炊，与《后汉书》"時且寒，賜以頭巾履襪"为同一手势（取薪=火、作糜=食、巾袜=衣，三件套两件在传一件在诗）。"待遇"之确解待议 |
| 思欲一**東歸** | 眷**東顧**兮但悲歎（其二） | 强：东向槽，隔山互望 |
| **行行日已遠** | **去去**割情戀，遄征**日遐邁**（其一） | 强：叠字+日+远同构 |
| **北風**聲正悲 | **北風**厲兮肅泠泠（其二） | 强 |
| **熊羆虎豹**夾路 | 出門無人聲，**豺狼**號且吠（其一） | 强：人去兽据槽 |
| 雪落霏霏／谿谷少人民 | 處所多霜雪／邊荒（其一） | 中强 |
| 人馬同時**飢** | **飢**當食兮不能餐（其二） | 中强 |
| 車輪**為之摧** | 車為不轉轍；匈臆**為摧**敗（其一） | 中："為…摧"句构 |
| 悠悠令我哀 | **悠悠**三千里（其一） | 中：悠悠槽 |

**棋谱定位**：《卻東西門行》是信（做雁勿做蓬、行止自成行有护送），《苦寒行》是**践约**——那支护送队在雪里走的样子。全谱更新：薤露/蒿里（编年史）→ 卻東西門行（劝归信，约206）→ **苦寒行（接归，206）** → 悲愤二首（誓问，207–208）→ 短歌行（应诺，约208）→ 善哉行（私信终章）。

## 四、【意外收获】曹叡《步出夏門行》：第三代读者

曹叡（曹操之孙、魏明帝）同调诗中（[古诗文网](https://m.gushiwen.cn/shiwenv_b124657f677f.aspx)、[国学梦](https://www.guoxuemeng.com/gushici/271919.html)）：

> ……迫日暮，**烏鵲南飛。繞樹三匝，何枝可依。**卒逢風雨，樹折枝摧。**雄來驚雌，雌獨愁棲。夜失羣侶，悲鳴徘徊。**……
> ……**孤禽失羣，悲鳴其間**……

曹叡将《短歌行》乌鹊四句**原文嵌入**，随即续写——并点明性别：**雌獨愁棲**。求贤读法中无性别的人才寓言，在曹操亲孙、最有条件知晓家族内情的读者笔下，是一只**失了群侣、独自愁栖的雌鸟**。另有"孤禽失羣，悲鳴其間"对其二"孤雁歸兮聲嚶嚶"。

**意义**：此为"乌鹊=她"读法的宫廷内部接受证据，与嵇康（嗟余薄祜）并列，构成第三代见证。又：曹叡名下另有以"薄祜"相关之乐府篇目待查（袁校对表所记"曹叡用例"之出处），**若坐实，曹叡同时持有"薄祐／祜"与"雌鸟"两枚密钥——他不是用例，是第三个读者。**待查项：曹叡该篇全文、出处（《乐府诗集》卷次）、早期文本作祐作祜。
